# 📖 Budget App <a name="about-project"></a>


**Budget App** is a rails app which lets users see and add their expense category, and expense

  </ul>

<!-- Features -->

### Key Features <a name="key-features"></a>


- **Display Splash page for unauthenticated users**
- **Register user**
- **login through a correct credentials**
- **Display Catagories list**
- **Display Expenses list**
- **Add new category**
- **Add new expense**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

<!--
Example command:

```sh
 gem install rails
```
 -->
  - Install ruby on your machine
  - Install rails on your machine

### Setup

Clone this repository to your desired folder:

```
git@github.com:zdnahom/budget-app.git
```
### Install

Install this project with:

```
cd budget-app
```



```
bundle install
```

### Usage

To run the project, execute the following command:

```
rails s
```

### Run tests

To run tests, run the following command:

```
rspec spec
```
